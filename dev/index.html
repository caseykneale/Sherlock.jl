Sherlock.inquire — Method
inquire( d::Detective, s::Union{Symbol,String} )

This function will search a loaded module for an instance or method of symbol s. If it finds it, it will, return an enumeration of its category. This is useful for quickly categorizing an item in a struct for display purposes.

Sherlock.sherlock_UI — Method
sherlock_UI()

This is a convenience function to create a Blink window which hosts an Interact UI. The intention of the UI is to explore a module (typed in by a user), and display some of the knowledge obtained about it graphically.
